The Workflow Simulator is designed to support MDI FHIR IG testing events. Each event, or other distinct set of tests, is collects any number of individual tests into a "module". Users will register with their email address and select the testing modules in which they wish to participate through the "Manage Registered Modules" option above. A report may then be generated for a given user based on their success for each test. Users may also view all available tests individually if they wish to run any outside of the scope of an event or other testing module.
