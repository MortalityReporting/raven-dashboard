Raven provides a series of interactive tests for users in alignment with test cases in alignment with MDI FHIR IG testing. Users may select individual test cases on their own as standalone tests, or register for testing events which contain a collection of tests being ran in that given event. For registered events, the status is tracked for review and support by event facilitators.
