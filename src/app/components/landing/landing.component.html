Welcome to the Raven Testing Platform!

Raven is a proof of concept for the Medicolegal Death Investigation (MDI) FHIR Implementation Guide (IG). It provides a tool for testing conformance to the MDI FHIR IG, including resource validation, record comparison, and data exchange workflows.

Raven also serves as an educational tool, providing examples FHIR data format (JSON and XML) and the ability to browse records and see how information is mapped between major fields and the underlying FHIR resources.
