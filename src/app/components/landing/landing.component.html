Record Viewer

View the different types of MDI records on the Raven FHIR Server and explore the underlying FHIR Resource structures. This includes both the MDI to EDRS Document Bundle and the Toxicology Report Message Bundle.

Import Case

Import a record to the Raven FHIR Server for use in other features.

FHIR Validator

Validate against the structure definitions for MDI FHIR Document Bundles or individual Resources and Profiles.

Record Comparison

Compare a user generated FHIR record against a known valid example from Raven.

Workflow Simulator

(Feature Under Development)

Move through steps of one of several test scenarios for various MDI related workflows, such as CMS to EDRS or a Toxicology Lab to CMS. The workflow simulator integrates other features.
